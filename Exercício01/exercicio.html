Como configurar o Git Bash no PC

Registrando seu Github no Git Bash

1º Entrar no Git Bash no seu computador.
2º Pesquisar no google 'Configuração incial Git'.
3º Entrar no Site e copiar os seguintes nomes para registrar no git bash.
$ git config --global user.name "Fulano de Tal"
$ git config --global user.email fulanodetal@exemplo.br
Lembrando, que o Username e User-email são referentes a sua conta no Git Hub.

Clonando seu repositorio

1º Entrar no seu Github.
2º Entrar no seu repositorio, escolher a pasta desejada e clicar no botão .CODE, após isso, abrirá
uma mensagem com um link em HTTP, escolha a opção HTML e clique em copiar.
3º Agora com o link copiado, entre no seu Git Bash e digite Git Clone + Link,
para clonar seu repositorio para o seu computador.
4º Caso queira confirmar, se a pasta foi baixada, digite no Git Bash - Git ls e localize sua pasta.
